Try vibe-vep directly in your browser — no installation required. This interactive
tutorial walks you through annotating cancer variants using the same engine that
powers the full CLI tool.

The demo includes KRAS and MRPL39 transcript data from GENCODE (GRCh38). The
annotation engine runs entirely in your browser via WebAssembly.
